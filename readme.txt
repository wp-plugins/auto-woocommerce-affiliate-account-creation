Automatically create affiliate accounts for your WooCommerce users.

== Description ==

This addon allows you to automatically create affiliate accounts for your WooCommerce users.

When customers create user accounts via WooCommerce, this addon will also create affiliate accounts for them.

This addon requires the [Affiliates Manager Plugin](https://wordpress.org/plugins/affiliates-manager/).

Read the usage documentation [here](https://wpaffiliatemanager.com/automatically-create-affiliate-account-woocommerce-customers/).

== Installation ==

1. Upload `wpam-auto-woocommerce-affiliate-creation` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
